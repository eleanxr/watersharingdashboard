Agriculture Data How-To

There are two ways to provide crop data. You can provide your own data using a Microsoft Excel spreadsheet, or you can automatically download data from the USDA National Agricultural Statistics Service. Once you have chosen a data source, it is recommended that you provide information about the price per acre, irrigation water requirement, and labor requirement for each crop using the crop grouping tool.

Custom Input with an Excel Spreadsheet

To provide data via an Excel spreadsheet, you must provide a spreadsheet containing the following columns:

  • Year: The year in which the crop was recorded.
  • Crop: The crop to which the measurement pertains.
  • Value: The measurement of acres or dollars for the crop
  • Unit: The unit of measurement. This should be either 'ACRES' or '$', to indicate that crop area or value of the harvested crop is being provided.

The spreadsheet may contain other columns, but those will not be used by the analysis. There should be one row entry for each crop measurement (consisting of a year, crop, value and unit). As an example, suppose you have measurements for a county in which corn, wheat, and hay are the primary crops, and you have 3 years of acreage data: 2000, 2005, and 2010. Then the input spreadsheet would have the following format:

Year Crop Value Unit
2000 Corn 10000 ACRES
2000 Wheat 20000 ACRES
2000 Hay 30000 ACRES
2005 Corn 5000 ACRES
2005 Wheat 25000 ACRES
2005 Hay 15000 ACRES
2010 Corn 8000 ACRES
2010 Wheat 12000 ACRES
2010 Hay 35000 ACRES

Once you've provided your data in the spreadsheet, some simple statistics will become available. To obtain more detailed statistics, edit the crop groups to provide estimates of the value of each crop per acre, the irrigation water requirement for each crop, and the number of hours of labor required to harvest the crop.

Using USDA NASS Data

In lieu of providing your own spreadsheet, you can choose to obtain your crop data automatically from the USDA National Agriculture Statistics Service (NASS). However, the NASS data is sometimes incomplete and sometimes double- or triple- counts certain crops, so additional analysis is required to assess the accuracy of the USDA data. To use the USDA NASS dataset, choose a source type of USDA NASS in the "Source Type" dropdown menu. Using the USDA NASS data source requires you to select three key parameters:

  • Source: the data source to use from the NASS dataset. Your choices are 'Census' or 'Survey'. Survey data are available for more years, but are less accurate and may contain information pertaining to fewer crops than the Census data. Census data are available only at 5 year intervals.
  • Analysis Years: The years for which to query the USDA NASS system. As noted above, certain kinds of data are available only for certain years. It is possible to leave this section empty, which will cause the system to obtain all available years of the selected data source. You may refine the set of years later by editing the crop mix.
  • Commodities: The set of commodities for which to query the USDA NASS system. It is possible to leave this section empty, which will cause the system to obtain all available commodities for the selected county. You may refine the set of commodities later.

Editing crop groups

The group edit interface allows you to provide additional information about the price, irrigation, and labor requirements of each crop. In addition, this interface allows you to group certain commodities together under a common category. To access the group edit interface, click on the "Edit groups" button on the main crop mix edit page.

Once in the crop group tool, you can add rows to the displayed table to assign values and commodities to different groups. The columns are as follows:

  • Group Name: Choose a name for the group. This will be displayed on all output results.
  • Revenue: This is the value, in dollars/acre, of the crops that are part of the group.
  • Labor: This is the number of hours/acre required to cultivate and harvest the crops assigned to the group.
  • NIWR: This is the net irrigation water requirement for the crops in the group, expressed in acre-feet per season.
  • Items: This is the list of commodities that should be considered as part of the group. The choices are provided by the input data, and you can choose multiple items for a single group.
